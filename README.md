# Exploratory Data Analysis (EDA) on HR Employee Attrition Dataset

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the HR Employee Attrition dataset to discover meaningful patterns, trends, and relationships within employee data.

The main objective is to understand the factors that influence employee attrition by using statistical summaries and various data visualization techniques.

---

## Objectives

- Understand the structure of the dataset.
- Perform data cleaning and preprocessing.
- Analyze statistical summaries.
- Identify missing and duplicate values.
- Explore data using visualizations.
- Discover relationships between different variables.
- Analyze correlations among numerical features.
- Generate business insights from the data.

---

## Dataset

The dataset contains employee-related information such as:

- Age
- Gender
- Department
- Job Role
- Monthly Income
- Education
- Marital Status
- Years at Company
- Overtime
- Attrition
- and several other employee attributes.
